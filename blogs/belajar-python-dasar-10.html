Belajar Python Dasar || Membuat Program Operasi Aritmatika

Di part sebelumya kita sudah belajar mamahami apa itu operasi aritmatika dan macam macam operasi aritmatika di python. Sekarang kita akan masuk ke seri untuk mendalami operasi aritmatika secara lebih lanjut yaitu dengan latihan operasi aritmatika.

Latihan Perhitungan Sederhana

Dalam sesi latihan aritmatika saya akan membagikan sedikit perhitungan sederhana dengan menggunakan aritmatika.

1. Menghitung Luas Segitiga

Oke pertama kita latihan mencari luas segitiga menggunakan kode program python. Let’s Gooo

I. Buka IDE Kalian

Jika kalian menggunakan IDE Pycharm atau IDE yang lain silahkan bisa dibuka. Untuk kali ini dan seterusnya saya akan menggunakan IDE PyCharm. Karena IDE tersebut menurut saya paling mudah digunakan.

II. Buat New Project

Jika IDE sudah terbuka, untuk yang mengunakan IDE PyCharm bisa klik File > New Project > Klik Create .

III. Buat New File

Setelah buat New Project, sekarang kita tinggal buat New File dengan cara Klik Kanan di pythonProject > Klik New > Klik Python File.

Ketikkan Nama Filenya (namanya bebas terserah kalian)

IV. Mulai Membuat Program

- Membuat Variabel

Rumus Segitiga adalah L = ½ x a x t.

Pertama - kita akan membuat variabel penampung ½. Untuk membuat ½ kita tidak bisa langsung menuliskan ½ tapi kita harus menuliskan 0.5 seperti gambar dibawah.

                
s = 0.5

Kedua – kita akan membuat variabel penampung panjang Alas. Untuk membuat variabel tersebut kita juga akan menerapkan Input User. Supaya kayak kalkulator gituu….

                
#Variabel untuk menampung alas
alas = int(input('Masukkan panjang alas : '))

mmm kenapa ada int() di depan input()? Itu karena kita akan menggunakan tipe data integer yang kita tau tipe data integer direprentasikan dengan int(). Kenapa integer?? Karena tipe data default dari input adalah string dan string tidak bisa di kalikan atau di tambahkan jadi kita harus ubah menjadi integer.

Ketiga – kita akan membuat variabel penampung tinggi segitiga. Untuk membuat variabel ini kurang lebih sama dengan alas segitiga. Kita bisa mengetikkan kode.

                
#variabel untuk menampung tinggi
tinggi = int(input('Masukkan Tinggi Segitiga : '))

Keempat – Selanjutnya kita akan membuat variabel penampung rumus segitiga. Nah untuk membuat variabel rumus nya kita tinggal pangil pangil variabel yang sudah dibuat. Kita bisa ketikkan : 

                
#variabel untul menampung luas segitiga
luas = s * alas * tinggi

Kelima – Nah sekarang kita tinggal tampilin hasil perkalianya deh… . Kita bisa mengetikkan :

                
#variabel untul menampung luas segitiga
luas = s * alas * tinggi

Keenam – Nah keenam kita bisa Run kodennya dengan cara Klik Kanan > Pilih Run.

Nah gimana mudahkan menerapkan operasi aritmatika…. .

V. Tampilan Semua Source Code 

                
#Program Untuk Menghitung Luas Segitiga
#Rumus Luas Segitiga = 1/2 x a x a

#Variabel untuk menampung 1/2
s = 0.5

#Variabel untuk menampung alas
alas = int(input('Masukkan panjang alas : '))

#variabel untuk menampung tinggi
tinggi = int(input('Masukkan Tinggi Segitiga : '))

#variabel untul menampung luas segitiga
luas = s * alas * tinggi

print('hasil perhitungan luas segitiga dengan alas = ', alas, 'dan tinggi = ', tinggi, 'adalah luas)
