Apa itu pemrograman ?

Apa itu pemrograman

Mungkin di masa-masa sekarang khususnya generasi milenial atau zaman Revolusi Industry 4.0 tidak asing dengan kata “Pemrogramman”. Baik di kesempatan kali ini aku akan sedikit membahas Apa Itu Pemrogramman.

Pemrograman adalah kegiatan atau proses menulis, menguji, memperbaiki, dan memelihara kode untuk suatu computer yang di tulis menggunakan Bahasa pemrograman (src : Wikipedia ). Lebih singkatanya adalah membuat program computer dengan kode. Aktivitas ini juga biasa atau bahkan sering di sebut ngoding atau coding.

manfaat pemrograman

jika kalian memilih untuk belajar pemrograamn itu adalah pilihan yang sangat sangat tepat. Karena, selain bidang pemrograman, sekarang banyak dicari di semua perusahaan / startup di seluruh dunia, kalian juga akan mendapatkan banyak manfaat lainya yaitu :

1. Skill yang terus berkembang

Di era industry 4.0 kemampuan atau sklill sangat dihargai. Karena skill dan kemampuan pasti diiringi dengan experience atau pengalaman dan di zaman sekarang lebih banyak di cari atau di utamakan seberapa besar pengalaman dalam melakukan kegiatan atau pekerjaan daripada besarnya teori yang dimiliki tanpa adanya pengalaman.

2. Melatih logika

Selain belajar pemrograman itu bisa mengembangkan skill kita. Dengan belajar pemrogramman juga bisa melatih logika lita lhoo.

Sebab, saat melakukan coding Bahasa pemrograman kita di tuntut untuk menggunakan logika kita untuk membuat baris kode program dan memcahkan masalah yang dialami saat melakuakan coding.

3. Menambah kreativitas

Disamping melatih logika tanpa kita sadari, pemrogramman juga menambah kreativitas. Karena, pemrograman juga membutuhkan suatu keratifitas dan imajinasi yang tinggi untuk membuat program yang rapih, bersih dan mudah di baca oleh orang lain.

4. Mempunyai uang tambahan/sampingan

Nah ini sisi terbaik dari belajar pemrograman yaitu peluang mempunyai uang tambahan. sebab dengan berbekal pemrograman kita bisa melakukan freelanc atau kerja online di suatu website yang menjualkan jasa pemrograman kita, dan penghasilan juga gak kalah besar tentunya dengan pekerjaan produktif kita lhoo

Jenis programmer dan Bahasa pemrograman

Dalam dunia pemrogramman paastinya mempunyai Bahasa yang berbeda dan kategori yang berbeda. Di sini aku akan coba merangkum kan kategori kategori Bahasa pemrograman dan contohnya

1. Web developer

Web developer adalah seorang programmer yang membuat program dan aplikasi berbentuk website (cth: shinaukhoding.ga). Dalam bidang web developer di bagi menjadi 2 kategoro yaitu:

a. Front End Developer

Sesuai namanya dari kata Front berarti depan. Jadi Front End Developer adalah seseorang yang mengatur atau mengurus tampilan depan sebuah website yang dilihat oleh pengunjung. Lantas Bahasa pemrogramman apa yang harus di kuasai oleh seorang Front End Developer

i. HTML( Hypertext Markup Languange )

HTML atau Hypertext Markup Languange adalah sebuah Bahasa untuk membuat struktur dalam sebuah website. Seperti Link, teks, paragraf dan lain lain

ii. CSS ( Cascading Style Sheet )

CSS atau Cascading Style Sheet adalah sebuah Bahasa yang selalu berdampingan dengan HTML yang memiliki kegunaan untuk memberikan style atau gaya atau tampilan dari element yang ada di HTML

iii. JavaScript

JavaScript adalah suatu Bahasa yang membuat sebuah website menjadi dinamis dan interaktif dan Bahasa ini juga sangat populer

b. Back End Developer

Setelah kita membahas apa itu Front End Developer, sekarang kita lanjutkan membahas apa itu Back End Developer.

Back End Developer adalah suatu kebalikan dari Front End developer, jika Front End Developer mengatur tampilan website makan Back End Developer adalah seorang yang mengatur kinerja atau mesin dari sebuah website.

Jadi tugas utama utama web developer adalah melakukan pemrosesan data seorang pengguna website tersebut. Terus Bahasa pemrograman apa yang di gunakan Back End Developer.

i. PHP
ii. Python
iii. Node.js
iv. SQL

Itu adalah Bahasa yang sering di gunakan sekarang oleh seorang Back End Developer. Dan kalian boleh pilih salah satu dari 4 bahasa pemrogramann di atas.

2. Software developer

Software developer adalah seorang yang membuat perangkat lunak yang berjalan di Windows, MacOS dan Linux. Bahasa pemrograman yang di pakai oleh seorang Software Developer adalah

i. Python
ii. Java
iii. C
iv. C++
v. C# dan masih banyak lagi

3. Mobile app developer

Kalua Software marancang perangkat lunak untuk Windows, macOS dan linux maka seorang Mobile App Developer memiliki tugas yaitu membuat atau merancang sebuah aplikasi untuk smartphone.

Dan Bahasa yang sering digunakan seorang mobile app developer yaitu:

i. C
ii. C++
ii. Python
iv. Java dan masih banyak lagi
