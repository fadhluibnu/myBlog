Syarat – Syarat Membuat Variabel

Sebelum kita belajat mulai membuat variabel. Kita harus tau syarat – syarat pembuatan variabel.

1. Variabel Terdiri dari huruf, angka dan karakter (underscore “_”)

Dalam pembuatan sebuah variabel kita bebas menggunakan apapun entah itu angka

• Contoh :

                
belajar_1 = ('hello world')
_belajar1 = ('hello world 2')
print(belajar_1)
print(_belajar1)

Maka jika kode di atas di run/di jalankan akan mengasilkan output

                
hello world
hello world 2

2. Diawali Huruf atau underscore ('_')

Meskipun kita boleh menggunakan huruf, angka dan karakter. Namun, ada yang tidak boleh digunakan saat pembuatan sebuah variabel.

• Contoh yang salah :

                
1contoh = ('hello world')
print(1contoh)

Jangan mengawali nama variabel dengan sebuah angka karena nanti saat di run akan terjadi error

• Contoh yang benar :

                
contoh1 = ('hello world')
print(contoh1)

Maka akan muncul output

                
hello world

3. Tidak menggunakan spasi

Selain angka dan lain lain, dalam pembuatan sebuah variabel juga tidak boleh menggunakan spasi

• Contoh yang salah :

                
contoh 1 = ('hello world')
print(contoh 1)

jika kalian menggunakan sepasi maka kode tersebut akan error saat di jalankan.

jika kalian menggunakan sepasi maka kode tersebut akan error saat di jalankan.

                
contoh_1 = ('hello world')
print(contoh_1)

Maka kalian harus menggunakan underscore untuk menisahkan angka dan huruf agar tidak error nantinya.

4. Tidak sama dengan keyword

Kita juga dilarang menggunakan keyword saat membuat variabel. Apa itu keyword?? Keyword adalah salah satu kata kunci yang sudah di buat oeh python untuk mendefinisiakan kode program.

• Contoh yang salah/menggunakan keyword :

                
while = ('hello world')
print(while)

contoh diatas akan menghasilkan error saat kita run karena menggunakan while untuk membuat nama variabel. Sedikit info while termasuk ke dalam keyword yang digunakan untuk membuat sebuah pengulangan pada bahasa python.

• Contoh Yang benar :

                
_while = ('hello world')
print(_while)

jika kalian menulikan variabel seperti di atas maka tidak akan error jika di run. Loh kenapa gak error kan ada kata while?? Sebab sebelum kata while ada underscore sehingga while tidak lagi terbaca sebagar keyword.

Oke guys sudah paham tentang syarat – syarat pembuatan variabel ??. Semoga kalian paham.

Variabel

Sesudah kita tau syarat – syarat pembuatan variabel. Nah sekarang kita mulai membuat variabel nih.

Hanya untuk mengingat Variabel adalah lokasi untuk menyimpan/menampung sebuah data di memori dalam sebuah program yang bisa berubah – ubah.

• Contoh Varibabel :

                
x = “hello guys”
print(x)

Maka akan menghasilkan

                
hello guys

huruf x diatas merupakan contoh sebuah deklarasi Variabel. Mmm Kenapa kok saat kita print(x) malah muncul hello guys?? Mmm ada yang tau??. Oke saya akan coba menjawab kenapa yang muncul hello guys.

Jadi seperti ini yang terjadi dibalik layar. Sebuah kalimat hello guys akan dimasukkan atau di tampung ke variabel x. Artinya, x itu sama dengan hello guys. Jadi saat kita print(x) maka otomatis yang tampil adalah hello guys karena x sama dengan hello guys.
