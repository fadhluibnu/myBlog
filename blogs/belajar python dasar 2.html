Belajar Python Dasar #2 || Cara Install Python Di Windows

Setelah kita tau apa itu python. Nah sekarang saya akan mebagikan bagaimana cara install python di windows

Download Python

Untuk mendapatkan file python kita bisa download di python.org

Install Python

1. Install Python File .exe

Buka file yang telah kalian download tadi dengan cara, klik kanan lalu pilih Run As Administrator

2. Setting Installasi

Disini kalian centang Install Launcer For All User (recommended) dan Add Python 3.9 to PATH. Setelah itu kalian pilih Install Now.

3. Proses Installasi

Kemudian akan muncul loading setup proses. Tunggu sampai selesai.

4. Selesai

Jika muncul Setup Was Succesful berarti proses install berhasil.

Coba Menjalankan Python

Untuk menjalankan python di laptop kalian sangatlah mudah, yaitu dengan cara :

1. Buka Command Prompt

Langkah pertama adalah buka command prompt kalian.

2. Mengetikkan Perintah

• Mengecek Versi Python

Untuk mengecek versi python kalian tinggal masukkan perintah

                
python --version

Maka jika muncul teks seperti di gambar berarti otomatis python kalian sudah ter-intasll di laptop kalian.

• Masuk Ke Python Shell Di CMD

Untuk masuk ke python shell di CDM kalian ketikkan perintah pada CMD

                
python

Maka akan mucul seperti gambar di bawah

3. Menulis Kode Python

Nah sekarang kita masuk ke menulis kode python sederhana. Kalian bisa ketikkan di CMD yang tadi

                
print("hello World")

maka akan muncul

                
hello World

seperti gambar di bawah
