Belajar Python Dasar #6 || Python Variabel Lanjutan

Sebelumnya kita sudah belajar tentang syarat syarat dalam pembuatan sebuah variabel. Sekarang kita akan melanjutakan seri khususnya python variabel yang sebelumnya.

Oke guys enjoy Lerning.

Membuat Variabel

Kita tau bahwa dalam pembuatan variabel memiliki syarat dan ketentuan. Nah sekarang kita akan belajar lebih lanjut tentang pembuatan variabel.

1. Menimpa Data

Penimpaan data akan terjadi saat kita membuat nama variabel dengan nama yang sama.

â€¢ Contoh

                
x = 12
x = 10
print(x)

Jika kita jalankan kode di atas maka akan muncul

                
10

Mmm kenapa bisa begitu? sebab, python itu di jalankan secara berurutan dari atas ke bawah, jadi saat kita run dan python menemukan x = 12 dan menemukan nama variabel yang sama yaitu x = 10 maka otomatis si nilai variabel x = 12 akan ter-timpa atau ter-upgrade oleh x = 10, oleh sebab itu saat kita jalankan print(x) kan muncul 10 karena x = 12 suda di timpa atau di upgrade oleh x = 10.

2. Assignment Variabel

Apa itu Assignment Variable?. Assignment Variable adalah sebuah pemberian data atau nilai dari variabel ke variabel. Mmm gimana tuhh??

Jadi Begini :

                
d = 20
n = d
print(n)

jika kita jalankan kode di atas maka akan mengasilkan

                
20

Mmm kenapa?? Sebuah variabel d itu sama dengan 20, jadi jika variabel n = d dan variabel d itu adalah 20 maka variabel n akan ikut menjadi 20.

3. banyak nilai ke beberapa variabel

Mmm maksudnya gimana?? Oke jadi sepert ini, dengan python memungkinkan kita memberikan beberapa nilai ke beberapa variabel dalam satu baris. Tapi yang perlu di ingat adalah kita harus membuat variabel dan value/ nilai jumlahnya harus sama.

â€¢ contoh :

                
x, y, z = 'jeruk',  'mangga',  'anggur'
print(x)
print(y)
print(z)

jika kode program di atas kita jalankan akan mengkasilkan :

â€¢ contoh :

                
jeruk
mangga
anggur

â€¢ contoh lain :

                
x, y = 'jeruk',  'mangga',  'anggur'
print(x)
print(y)

jika kita program di atas maka akan error. Kenapa? Karena jumlah variabel dan jumlah nilai /value berbeda. Oke jadi ingat ya, jumlah variabel dan value/nilai harus sama jika ingin menggunakan metode di atas.
